# 🌦 IoT Weather Monitoring & Rain Prediction (ESP32 + Blynk + ML)

An intelligent IoT system that monitors environmental conditions and predicts rainfall using Machine Learning models.

---

## 📌 Overview

This project combines **IoT (ESP32 + Blynk)** with **Machine Learning** to:

- Collect real-time weather data
- Analyze environmental conditions
- Predict rain using ML models

---

## 🎯 Objectives

- Real-time weather monitoring
- Cloud-based data visualization
- Predict rainfall using ML models
- Integrate IoT with Data Science

---

## 🧰 Technologies Used

### 🔹 Hardware
- ESP32
- Temperature Sensor
- Humidity Sensor
- Pressure Sensor (optional)

### 🔹 Software
- Python (Jupyter Notebook)
- Blynk Cloud
- Scikit-learn (ML models)
- Matplotlib / Seaborn

---

## ☁️ IoT Architecture

Sensors → ESP32 → Blynk Cloud →  
Jupyter Notebook → ML Model → Prediction

---

## 📊 Features

- Real-time weather data collection
- Data visualization (graphs)
- Machine learning classification:
  - Decision Tree
  - Support Vector Machine (SVM)
  - K-Nearest Neighbors (KNN)
- ROC Curve analysis
- Live prediction using Blynk data

---

## 🤖 Machine Learning Workflow

1. Data collection (Blynk / simulated)
2. Data preprocessing
3. Feature selection:
   - Temperature
   - Humidity
   - Pressure
   - Altitude
4. Model training
5. Model evaluation (Accuracy, ROC)
6. Real-time prediction

---

## 🌧️ Rain Prediction Logic

The model predicts rain based on:
- High humidity → higher chance of rain
- Pressure drop → possible rainfall
- Temperature variation → weather change

---

## 📈 Output

- Classification result (Rain / No Rain)
- Accuracy score
- ROC Curve visualization

---

## ▶️ How to Run

1. Connect ESP32 to Blynk Cloud
2. Upload sensor data
3. Open Jupyter Notebook
4. Run all cells
5. View prediction results
